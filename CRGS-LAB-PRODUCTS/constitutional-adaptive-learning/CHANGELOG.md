# Changelog

All notable changes to the Constitutional Adaptive Learning Systems Protocol (CALSP) research template will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [4.3.0] - 2026-06-23

### Added
- **Full Audit Remediation:** Comprehensive IRB compliance language throughout all documentation
- **Detailed Megaproject Budget:** Line-item budget with full justification ($7.2M over 36 months)
- **Falsifiable Hypotheses:** Eight explicit, testable hypotheses (H1-H8) linked to SEM
- **Global Hub Specifications:** Detailed definitions and recruitment quotas for four cultural hubs
- **Expanded Broader Impacts:** Comprehensive section on neurodivergent-first governance as model for inclusive AI
- **Project Summary Reformatting:** NSF-compliant format with explicit Intellectual Merit and Broader Impacts headings
- **Citations Management:** Clarified Zotero group library and comprehensive bibliography
- **Power Analysis Update:** Incorporated pilot effect sizes amendment pathway
- **GitHub Repository Template:** Complete open-source repository structure with all supporting files

### Changed
- **IRB Language Correction:** Bootstrapped pilot now explicitly requires institutional IRB exemption (45 CFR 46.104(d)(3))
- **CHAE Repositioning:** Clarified as supplemental oversight layer that augments (not replaces) institutional IRB
- **Ethics Checklist Redesignation:** Minimal Ethics Checklist now explicitly labeled as pre-submission planning tool
- **Budget Justification:** Added detailed personnel roles, equipment rationale, and F&A calculation
- **CALT Expression:** Theory now expressed through eight falsifiable hypotheses with clear operationalization
- **Recruitment Strategy:** Added specific sampling quotas and stratification criteria for each hub

### Fixed
- Removed any language suggesting CHAE replaces IRB authority
- Corrected timeline to show IRB exemption required before pilot data collection
- Added explicit statement that no data collection begins without documented IRB approval
- Clarified that CHAE's binding authority applies to operational governance, not human subjects protocols

### Governance
- All changes reviewed and approved by Neurodivergent Advisory Board (NAB)
- CHAE operating procedures updated to reflect supplemental oversight role
- Community Governance Board (CGB) ratified budget structure
- Open-Source Stewardship Committee (OSSC) approved repository architecture

## [4.2.0] - 2025-12-15

- **Bootstrapped Prototype Deployment Requirement:** Mandatory live demo before megaproject funding

### Fixed

- Improved timeline integration between solo and team phases

## [4.1.0] - 2025-09-01

- **Patch version (0.0.X):** Bug fixes, clarifications, or minor documentation updates
