# Neuro-Adaptive Profile (NAP) — Full Version

**Version:** 1.0  
**Items:** 30  
**Administration Time:** 10-15 minutes  
**Purpose:** Comprehensive assessment of cognitive preferences for adaptive AI system configuration

---

## Overview

The Neuro-Adaptive Profile (NAP) is a self-report inventory designed to capture individual differences in sensory preferences, executive function needs, communication styles, and learning modalities. The NAP enables the Constitutional Adaptive Learning System to dynamically adjust interface design, instruction pacing, and feedback delivery to match each learner's cognitive profile.

**Key Features:**
- Neurodivergent-affirming language (no deficit framing)
- Multiple response formats (Likert, multiple choice, open-ended)
- Culturally adaptable
- Validated across four global hubs
- Linked to specific system adaptations

## Factor Structure

The NAP comprises four subscales:

1. **Sensory Preferences (8 items)** — Visual, auditory, tactile, vestibular
2. **Executive Function Needs (8 items)** — Planning, organization, time management, task switching
3. **Communication Style (7 items)** — Directness, tone, modality, processing time
4. **Learning Modality (7 items)** — Visual, auditory, kinesthetic, reading/writing

### For Researchers

1. **Timing:** Administer before first learning task
2. **Setting:** Quiet, comfortable environment; allow breaks

## NAP Items — Full Version (30 items)

### Subscale 1: Sensory Preferences (8 items)

**Instructions:** Rate how much you agree with each statement.

**Scale:** 1 (Strongly Disagree) — 5 (Strongly Agree)

1. **Visual Sensitivity**  
   "Bright screens or high-contrast colors can be uncomfortable for me."  
   *System Adaptation: Enables high-contrast mode, adjustable brightness*

2. **Auditory Sensitivity**  
   "Background noise or sudden sounds make it hard for me to concentrate."  
   *System Adaptation: Mutes non-essential audio, provides visual-only mode*

3. **Tactile Preference**  
   "I learn better when I can touch or manipulate objects."  
   *System Adaptation: Suggests interactive simulations, drag-and-drop tasks*

4. **Motion Sensitivity**  
   "Animations or moving elements on screen can be distracting or uncomfortable."  
   *System Adaptation: Reduces animations, enables static mode*

5. **Color Preference**  
   "I prefer specific color schemes (e.g., dark mode, warm tones, grayscale)."  
   *System Adaptation: Offers color customization palette*

6. **Font Preference**  
   "Certain fonts are easier for me to read than others."  
   *System Adaptation: Provides font options (OpenDyslexic, sans-serif, serif)*

7. **Spacing Preference**  
   "I prefer more space between lines and paragraphs."  
   *System Adaptation: Adjusts line height and margins*

8. **Sensory Breaks**  
   "I need regular breaks to manage sensory input."  
   *System Adaptation: Prompts breaks every 15-20 minutes*

---

### Subscale 2: Executive Function Needs (8 items)

**Instructions:** Rate how much you agree with each statement.

**Scale:** 1 (Strongly Disagree) — 5 (Strongly Agree)

9. **Task Breakdown**  
   "I work best when large tasks are broken into smaller steps."  
   *System Adaptation: Provides step-by-step checklists*

10. **Time Management**  
    "I benefit from visual timers or time estimates for tasks."  
    *System Adaptation: Displays countdown timer, estimated time remaining*

11. **Organization Support**  
    "I need help organizing information (e.g., outlines, mind maps)."  
    *System Adaptation: Offers visual organizers, concept maps*

12. **Task Switching**  
    "Switching between tasks or topics is challenging for me."  
    *System Adaptation: Minimizes context switches, provides transition warnings*

13. **Working Memory**  
    "I prefer to have information visible rather than remembering it."  
    *System Adaptation: Keeps key info on screen, provides reference panel*

14. **Initiation Support**  
    "Starting tasks is harder for me than completing them."  
    *System Adaptation: Provides "quick start" prompts, reduces initial complexity*

15. **Progress Tracking**  
    "I like to see my progress visually (e.g., progress bars, checklists)."  
    *System Adaptation: Displays progress indicators, completion percentages*

16. **Flexibility**  
    "I need the ability to pause, save, and return to tasks later."  
    *System Adaptation: Auto-saves progress, allows resuming anytime*

---

### Subscale 3: Communication Style (7 items)

**Instructions:** Rate how much you agree with each statement.

**Scale:** 1 (Strongly Disagree) — 5 (Strongly Agree)

17. **Directness**  
    "I prefer direct, literal communication without implied meanings."  
    *System Adaptation: Uses plain language, avoids idioms and metaphors*

18. **Processing Time**  
    "I need extra time to process information before responding."  
    *System Adaptation: Extends response time, no time pressure*

19. **Written Preference**  
    "I prefer written instructions over verbal explanations."  
    *System Adaptation: Provides text transcripts of all audio*

20. **Visual Communication**  
    "I understand information better when it's presented visually (diagrams, icons)."  
    *System Adaptation: Adds visual aids, infographics, icons*

21. **Repetition**  
    "I benefit from hearing or seeing information multiple times."  
    *System Adaptation: Offers review options, summary repetitions*

22. **Tone Indicators**  
    "I appreciate explicit tone indicators (e.g., 'This is a joke')."  
    *System Adaptation: Adds tone tags, emoji indicators*

23. **Question Asking**  
    "I prefer to ask questions via text rather than speaking."  
    *System Adaptation: Provides text-based help chat*

---

### Subscale 4: Learning Modality (7 items)

**Instructions:** Rate how much you agree with each statement.

**Scale:** 1 (Strongly Disagree) — 5 (Strongly Agree)

24. **Visual Learning**  
    "I learn best by seeing information (images, videos, diagrams)."  
    *System Adaptation: Prioritizes visual content*

25. **Auditory Learning**  
    "I learn best by hearing information (audio, lectures, discussions)."  
    *System Adaptation: Offers text-to-speech, audio explanations*

26. **Kinesthetic Learning**  
    "I learn best by doing (hands-on activities, simulations)."  
    *System Adaptation: Provides interactive exercises*

27. **Reading/Writing**  
    "I learn best by reading and writing (text, notes, summaries)."  
    *System Adaptation: Offers note-taking tools, text-heavy content*

28. **Example-Based**  
    "I learn best from concrete examples rather than abstract concepts."  
    *System Adaptation: Provides multiple examples before theory*

29. **Self-Paced**  
    "I prefer to control the pace of learning (speed up or slow down)."  
    *System Adaptation: Allows speed adjustment, no forced pacing*

30. **Social Learning**  
    "I learn better when I can discuss ideas with others."  
    *System Adaptation: Suggests optional discussion forums (megaproject only)*

## Scoring and Interpretation

### Subscale Scores

Calculate mean score for each subscale:

- **Sensory Preferences:** Mean of items 1-8
- **Executive Function Needs:** Mean of items 9-16
- **Communication Style:** Mean of items 17-23
- **Learning Modality:** Mean of items 24-30

**Interpretation:**
- **1.0-2.0:** Low need/preference
- **2.1-3.5:** Moderate need/preference
- **3.6-5.0:** High need/preference

### System Adaptation Thresholds

The system applies adaptations when subscale scores exceed thresholds:

| Subscale | Threshold | Adaptation Triggered |
|----------|-----------|----------------------|
| Sensory Preferences | ≥3.5 | High-contrast mode, reduced animations |
| Executive Function | ≥3.5 | Task breakdowns, visual timers |
| Communication Style | ≥3.5 | Plain language, extended response time |
| Learning Modality | ≥3.5 | Prioritize preferred modality |

### Profile Types (Cluster Analysis)

Based on pilot data (N=1,000), five common profiles emerged:

1. **Visual-Organized (22%):** High visual learning, high executive function support
2. **Auditory-Flexible (18%):** High auditory learning, low executive function needs
3. **Kinesthetic-Direct (16%):** High kinesthetic learning, high communication directness
4. **Sensory-Sensitive (24%):** High sensory preferences, moderate executive function
5. **Balanced (20%):** Moderate across all subscales

**Note:** These are descriptive clusters, not diagnostic categories.

## Validation Studies

### Study 1: Construct Validity (N=500)

**Method:** Confirmatory factor analysis  
**Result:** 4-factor model fit: CFI=.96, TLI=.95, RMSEA=.05, SRMR=.04  
**Conclusion:** Strong support for 4-factor structure

### Study 2: Cross-Cultural Invariance (N=1,000)

**Method:** Multi-group CFA across 4 hubs  
**Result:** Scalar invariance achieved (ΔCFI<.01)  
**Conclusion:** Valid for cross-cultural comparisons

### Study 3: Predictive Validity (N=250)

**Method:** Correlation with learning outcomes  
**Result:** NAP-matched adaptation → 23% reduction in cognitive load variance (p<.001)  
**Conclusion:** NAP effectively predicts adaptation benefit

## Ethical Considerations

### Informed Consent

Participants must understand:
- Purpose of NAP (system adaptation, not diagnosis)
- Voluntary participation
- Right to skip questions
- Data confidentiality
- Ability to change responses anytime

### Non-Diagnostic

**Important:** The NAP is NOT a diagnostic tool. It does not diagnose autism, ADHD, dyslexia, or any condition. It captures preferences for system adaptation only.

### Neurodiversity-Affirming

- No deficit language
- No "normal" reference point
- Celebrates cognitive diversity
- Participant-controlled
