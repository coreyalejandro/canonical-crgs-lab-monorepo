III. GAP-02 — Fix core/boot.py Tautological Integrity Check (HIGH)

GAP-02 CONSTITUTION_HASH is computed from the live string — the integrity check is a self-confirming tautology HIGH

Root cause. The current code sets CONSTITUTION_HASH = hashlib.sha256(TLC_TOCA_CANON.encode("utf-8")).hexdigest(). Because the hash is derived from the string at runtime, any mutation of TLC_TOCA_CANON also changes CONSTITUTION_HASH in the same execution — the two values always match. Article 8's "hard crash on mutation" guarantee is void.

Fix. Pre-compute the hash of the authoritative constitution text and hard-code it as a string literal. Any subsequent mutation of the text will then produce a different runtime hash, which will not equal the literal, causing sys.exit(1) as intended.

end   = src.index('"""', start)
canon = src[start:end]
